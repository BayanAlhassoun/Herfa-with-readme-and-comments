TestMethods Folder
This folder contains the test method classes that are responsible for conducting various tests within the application. Each test class focuses on specific functionalities and ensures that the application behaves as expected.

Examples:

UserRegister_TestMethods.cs
Purpose: Contains tests for the user registration process.
Description: This class verifies that the registration flow works correctly. It tests various scenarios, such as successful registrations, input validation errors, and duplicate email handling. Each method aims to ensure that the user registration functionality meets the specified requirements.
