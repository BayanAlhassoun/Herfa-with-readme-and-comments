Helpers Folder
The Helpers folder contains essential classes that provide utility and helper methods to facilitate various functionalities within the application. These classes promote code reusability, maintainability, and modularity by encapsulating common operations and driver management. Below is a brief overview of the included classes:

Class Overview
CommonMethods.cs
Description: This class contains a collection of static helper methods that can be used across the application to perform common tasks, such as reading data from Excel files, Highlight elements, and other utility functions.
Purpose: To centralize reusable methods, reducing code duplication and improving maintainability.
ManageDriver.cs
Description: This class manages the browser driver used for automated testing. It includes methods for initializing the driver, navigating to different URLs, and closing the driver session.
Purpose: To streamline the process of driver management, ensuring consistent and efficient browser interactions during automated testing.
